# Portfolio v2

Personal portfolio website built with Next.js 15, React 18, TypeScript, and Tailwind CSS v4.

## Sections

- **Hero** — introduction and main call to action
- **Services** — services offered
- **Tech Stack** — technologies and tools
- **Experience** — work history
- **Projects** — featured projects with modals
- **SUNAT Billing** — specialized billing integration showcase
- **Contact** — contact form
- **Footer**

## Tech Stack

- [Next.js 15](https://nextjs.org/)
- [React 18](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Motion](https://motion.dev/) — animations
- [Radix UI](https://www.radix-ui.com/) — accessible UI primitives
- [shadcn/ui](https://ui.shadcn.com/) — component library
- [MUI](https://mui.com/) — Material UI icons and components
- [Lucide React](https://lucide.dev/) — icons
- [Recharts](https://recharts.org/) — charts

## Getting Started

Install dependencies:

```bash
npm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

| Command         | Description                |
|-----------------|----------------------------|
| `npm run dev`   | Start development server   |
| `npm run build` | Build for production       |
| `npm run start` | Start production server    |
| `npm run lint`  | Run ESLint                 |
